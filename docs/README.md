# Documentation

Ce dossier centralise les documents du projet ArtiBot.

## Fichiers

- `ArtiBot_Cahier_des_Charges.docx`
  CDC produit initial.
- `ArtiBot_CDC_Technique.docx`
  CDC technique du MVP.
- `SUIVI.md`
  État d'avancement du projet.
- `EVOLUTIONS_PRODUIT_NAV.md`
  Recommandations concrètes sur ce qu'il manque dans le produit et la navigation.
- `PLAN_MVP_VENDABLE.md`
  Liste priorisée de ce qu'il faut ajouter pour rendre ArtiBot réellement vendable en pilote puis en SaaS.

## Usage

- Lire `SUIVI.md` pour l'état courant.
- Lire `EVOLUTIONS_PRODUIT_NAV.md` pour les prochaines améliorations UI/produit.
- Lire `PLAN_MVP_VENDABLE.md` pour les priorités produit/tech avant commercialisation.
- Garder les futurs documents projet dans ce dossier.
